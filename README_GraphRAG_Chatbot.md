# GAIS GraphRAG Chatbot - AI使用上の法的リスク共有プラットフォーム

## 🎯 概要

GAIS（生成AI協会）会員向けに開発された、AI使用上の法的リスクに関する知識を共有し、安心してAI開発に取り組むための専門的なチャットボットシステムです。

### 主な特徴

- 📄 **PDFアップロード機能**: 最新の法的資料やガイドラインをリアルタイムで学習
- 🔍 **GraphRAG技術**: アップロードされた資料をグラフデータベースで構造化し、高度な検索を実現
- 💬 **専門的なAIアシスタント**: Claude 3.5を活用した法的リスクに特化した対話型AI
- 🌐 **最新情報の統合**: Web検索機能により、最新の法規制や判例を即座に取得
- 👥 **会員限定アクセス**: GAIS会員のみがアクセス可能なセキュアな環境

## 🚀 使い方

### 1. PDFアップロード

1. GAIS会員のメールアドレスを入力
2. **「📄 PDF Upload」** ボタンをクリック
3. AI法務関連のPDF資料（最大10MB）を選択してアップロード
4. アップロードされた資料は自動的に解析され、GraphRAGデータベースに保存

### 2. 法的リスクに関する質問

```
例：
- 「生成AIで作成したコンテンツの著作権はどう扱われますか？」
- 「AI生成画像を商用利用する際の法的リスクは？」
- 「個人情報保護法とAI開発の関係について教えてください」
- 「ディープフェイク技術の法的規制について」
```

### 3. 検索オプション

- ✅ **Graph RAG検索**: アップロードされたPDF資料から関連情報を検索
- ✅ **Web検索 (Tavily)**: 最新のWeb情報から法規制や判例を検索

### 4. 会員ダッシュボード

メールアドレスを入力すると **「📊 Dashboard」** ボタンが表示されます。クリックすると以下の情報を確認できます：

- 📊 アップロードした資料の統計情報を表示
- 📈 総ページ数、分析チャンク数などの詳細データ
- 📋 最近のアップロード履歴
- 👤 会員情報（メールアドレス、所属組織）

## 📚 GraphRAGの仕組み

### 1. PDF解析プロセス

```
PDF → テキスト抽出 → チャンク分割 → エンティティ抽出 → グラフ構築
```

### 2. グラフ構造

```
(Document) -[:CONTAINS]-> (Chunk)
    │
    └─[:MENTIONS]-> (Entity: 法的概念)
    │
    └─[:UPLOADED_BY]-> (Member: GAIS会員)
```

### 3. 検索アルゴリズム

1. **ベクトル類似度検索**: 質問と関連するチャンクを検索
2. **グラフトラバーサル**: 関連エンティティを通じて文脈を拡張
3. **ランキング**: 関連性スコアに基づいて結果を順位付け
